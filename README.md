# Student Management System (C++)

A simple console-based Student Management System built in C++ using Object-Oriented Programming (OOP) and File Handling.

This project allows users to manage student records through a menu-driven interface and stores the data permanently in a text file.

## Features

- Add Student
- Display All Students
- Search Student by Roll Numbers
- Prevent duplicate roll numbers
- Update Student Details
- Delete Student Record
- Persistent storage using `student.txt`
- Menu-driven interface

## Concepts Used

- Object-Oriented Programming (OOP)
- Encapsulation
- Classes and Objects
- Getters and Setters
- File Handling (`ifstream` and `ofstream`)
- CRUD Operations
- Functions
- Loops and Conditional Statements

## How to Run

1. Clone the repository

```bash
git clone <repository-link>
```

2. Open the project in Visual Studio Code.

3. Compile the program

```bash
g++ main.cpp -o student
```

4. Run the executable

```bash
./student
```

(Windows)

```bash
student.exe
```
